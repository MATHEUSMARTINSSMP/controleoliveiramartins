# Guia do Usuário - Integração Google My Business

Este guia explica como utilizar as funcionalidades da integração com o Google My Business no Painel Administrativo.

## 1. Conexão
1.  Acesse a aba **Google Integration**.
2.  Clique no botão **Conectar Conta Google**.
3.  Você será redirecionado para o Google. Faça login e autorize as permissões solicitadas.
4.  Ao retornar, você verá o status "Conectado" e sua foto de perfil.

## 2. Gerenciando Reviews
*   **Visualizar**: Na aba "Reviews", você vê todos os comentários recebidos. Use os filtros para buscar por data, nota ou status.
*   **Responder**: Clique no botão "Responder" em um card de review.
    *   Escreva sua resposta ou escolha um **Template**.
    *   Use o botão **Gerar com IA** para criar uma sugestão automática.
*   **Marcar como lida**: Reviews novos aparecem com uma etiqueta "Nova". Clique no ícone de "check" para marcar como lida.

## 3. Estatísticas
Acesse a aba "Estatísticas" para ver:
*   **Evolução**: Gráfico de reviews ao longo do tempo.
*   **Distribuição**: Gráfico de pizza mostrando a proporção de notas (1 a 5 estrelas).
*   **Insights**: Palavras-chave mais citadas e análise de sentimento.

## 4. Postagens
Acesse a aba "Postagens" para criar conteúdo no seu perfil Google:
1.  Clique em **Nova Postagem**.
2.  Escolha o tipo (Novidade, Evento, Oferta).
3.  Escreva o conteúdo e adicione um botão de ação (opcional).
4.  Clique em **Publicar**.

## 5. Configurações
Na aba "Configurações" (ícone de engrenagem nas notificações), você pode:
*   Ativar/desativar alertas de novos reviews.
*   Configurar alertas específicos para reviews negativos.
